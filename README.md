# DARK SKY — pds

**"They have always been here."**

A single-file browser game. You are the UAP. The tape begins in 1908 and ends at the door.

- 8-tasking campaign spanning 1908 → 2026, with era-correct enemies, film stocks, and comms
- Era checkpoints — resume the decade you fell in
- FLIR sensor-feed presentation with white/black-hot polarity (2004+)
- Optional satellite ground feed (real overhead imagery, FLIR-processed)
- Desktop (keyboard / mouse / gamepad) and mobile (portrait or landscape, touch controls)

## Run it

Open `index.html` in any browser, or enable GitHub Pages on this repo
(Settings → Pages → branch `main`, root) and play at the published URL.

No build step. No dependencies. One file.
